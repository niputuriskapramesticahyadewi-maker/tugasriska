1.Konsep Array
Program menggunakan list di Python sebagai representasi array untuk menyimpan nilai mahasiswa.
-Array memudahkan akses data dengan indeks dan operasi seperti mencari maksimum, minimum, rata-rata, serta menghitung jumlah mahasiswa lulus/tidak lulus.

2.Hasil Eksekusi
Output teks:

Kode
Nilai: [70, 61, 55, 80, 90, 75, 78, 81, 40, 95]
Nilai Tertinggi: 95
Nilai Terendah: 40
Rata-rata: 72.5
Jumlah Lulus: 8
Jumlah Tidak Lulus: 2
Grafik batang:
Nilai tertinggi (biru) vs terendah (merah).
Jumlah mahasiswa lulus (hijau) vs tidak lulus (orange).

3.Analisis Kompleksitas
Time complexity: O(n), karena operasi max, min, sum, dan perulangan kelulusan semuanya linear terhadap jumlah data.
Space complexity: O(n), karena list menyimpan semua nilai mahasiswa.

4.Refleksi Pembelajaran
-Belajar dasar penggunaan array/list untuk menyimpan dan mengolah data.
-Menerapkan operasi analisis sederhana (max, min, rata-rata, kondisi lulus).
-Memvisualisasikan hasil dengan grafik agar lebih mudah dipahami.
-Memahami pentingnya analisis kompleksitas agar program tetap efisien jika jumlah data bertambah besar.
